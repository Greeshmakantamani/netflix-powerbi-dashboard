# Netflix Power BI Dashboard

## Overview
This project analyzes Netflix content using Power BI.

## Features
- Genre analysis
- Country-wise content trends
- Ratings analysis
- Release year insights
- Interactive filters and slicers

## Tools Used
- Power BI
- DAX
- CSV Dataset
